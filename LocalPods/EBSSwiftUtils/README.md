## Installation

EBSSwiftUtils is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'EBSSwiftUtils', :git => 'https://git2.devebs.net/ebs-mobile/ios/swift-utils.git'
```
